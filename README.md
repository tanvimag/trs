# Train Reservation System 🚆

A console-based Train Reservation System developed using the C programming language.  
This project simulates a real-world railway booking platform where users can reserve tickets, view train details, validate routes, and manage reservations efficiently.

---

## 📌 Project Overview

The Train Reservation System (TRS) is designed to automate railway ticket booking operations through a menu-driven interface.  
The system supports multiple trains, station validation, seat class selection, ticket booking, and reservation management.

This project demonstrates the implementation of:
- File handling
- Structures in C
- Arrays and functions
- Data validation
- Menu-driven programming
- Real-world problem solving using C

## ⚙️ Installation & Execution

### Clone the Repository

```bash
git clone https://github.com/tanvimag/trs.git
```

### Navigate to the Project Directory

```bash
cd trs
```

### Compile the Program

```bash
gcc train_reservation.c -o trs
```

### Run the Program

#### Linux / macOS

```bash
./trs
```

#### Windows

```bash
trs.exe
```
